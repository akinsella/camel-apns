Using camel-apns
===========
 
Please, refer to following wiki related page: http://code.google.com/p/camel-apns/

==============
Building camel-apns
==============

camel-apns is built with Maven 2. To rebuild camel-apns-VERSION.jar,
place Maven 2 on the PATH and execute "mvn package".    


In case of problems send an e-mail note to: dev@helyx.org.
